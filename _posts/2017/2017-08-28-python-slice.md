```python
>>> L = ['Michael', 'Sarah', 'Tracy', 'Bob', 'Jack']
>>> L[0:3]  # 从索引0开始取，直到索引3为止，但不包括索引3。
['Michael', 'Sarah', 'Tracy']
>>> L[:3] # 从索引0开始取，直到索引3为止，但不包括索引3。这里省略了0.
['Michael', 'Sarah', 'Tracy']
>>> L[1:3]
['Sarah', 'Tracy']
>>> L[-2:]
['Bob', 'Jack']
>>> L[-2:-1]
['Bob']
```
