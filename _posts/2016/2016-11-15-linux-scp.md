### 3 ways to transfer file between linux

1. ftp

2. samba

3. scp


```
copy file 'testfile' to remote linux.
scp testfile jian@172.17.92.240:/home/jian
```

```
scp -P 2200 testfile jian@172.17.92.240:/home/jian
testfile 和 -P 2200 的位置不能对掉，有点坑
```
